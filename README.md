# Dolly — Your AI Film Crew

Brand identity showcase website for **Dolly**, an autonomous AI film crew. Three cameras. One brain. Zero crew.

Built for **AdventureX Hangzhou 2026**.

## Overview

This is a single-page brand identity website that recreates the [Dolly Brand Identity – Version 2](./Dolly%20Brand%20Identity-Version-2.pdf) document as a modern, responsive web experience. It covers all 10 sections of the brand system:

1. **Logo System** — Primary lockup, horizontal lockup, app icon, monogram
2. **Color** — Midnight, Signal, Glow, Ivory, Ink palette
3. **Typography** — Archivo + IBM Plex Mono
4. **The Four Buttons** — Action, Again, Hold, Cut voice commands
5. **Voice** — Brand tone and cadence guidelines
6. **Imagery** — Photography and visual style
7. **Deck System** — Slide presentation templates
8. **Brand Graphics** — Switch diagram, pixel mark, boot screen, sticker sheet
9. **Meet Dolly** — The mascot character
10. **Poster Series** — Campaign posters

## Tech Stack

- Pure HTML + CSS + vanilla JavaScript
- Google Fonts: Archivo, IBM Plex Mono
- No build tools, no frameworks, no external dependencies
- Fully responsive, mobile-first design

## Local Development

Just open `index.html` in any browser:

```bash
# No build step required — open directly
start index.html
```

## Deployment

Push to GitHub and enable GitHub Pages from the root of the `main` branch:

1. Go to repo **Settings → Pages**
2. Source: **Deploy from a branch**
3. Branch: **main**, folder: **/ (root)**
4. Site will be live at `https://<username>.github.io/<repo>/`
